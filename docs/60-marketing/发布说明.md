# 发布说明

## 2026-07-15：组织管理与组织账单正式上线

eLight 已上线组织管理与组织账单能力，帮助企业和团队在不改变个人账号扣费方式的前提下，集中管理成员并分析 AI 消费。

### 主要能力

- 系统管理员创建和维护组织，组织 Admin 管理成员与角色。
- Admin 查看组织完整账单，Member 查看本人在组织有效期内的用量。
- 按成员、模型、渠道和日期分析消费金额、原始 quota、请求数与 Token 用量。
- 提供趋势、消费明细和账单导出，保留可审计的历史消费事实。
- 支持组织成员加入、离开和历史有效期口径，组织外消费不会混入组织账单。

### 兼容性说明

个人账号仍是唯一扣费主体。组织不持有余额、订阅、充值订单或 API Key，也不参与请求预扣、结算和退款；现有个人钱包、订阅、Token 和日志链路保持不变。
